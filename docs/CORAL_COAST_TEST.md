# Coral Coast Test

- [ ] Enter Coral Coast through the Academy Coral Coast Gate.
- [ ] Finn Tide welcomes the player and explains the shell mission.
- [ ] Passport explains that Lighthouse Hero unlocks the Coral Coast stamp.

- [ ] Discovery Beach is safe to explore with arrow keys.
- [ ] Seashells are easy to see and collect.
- [ ] Educational coral and seashell signs are readable.
- [ ] Five collected seashells complete the shell objective.
- [ ] Ocean Explorer badge and stickers unlock.

## Wave Rider Raceway

- [ ] Track is wide and beginner-friendly.
- [ ] Coral arches and ocean tunnels are visible.
- [ ] Seashell counter updates during the race.
- [ ] Lighthouse clearly marks the finish.
- [ ] Wave Rider sticker unlocks after finishing.
- [ ] Lighthouse Hero completes after all three objectives.
