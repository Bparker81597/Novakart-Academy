# Character Introduction Test

Run this checklist with a child and avoid explaining the screen unless they
become stuck.

## Character Checks

- [ ] Select Blaze Bolt. Blaze's orange/red theme and lightning intro appear.
- [ ] Select Finn Tide. Finn's blue/teal theme and ocean intro appear.
- [ ] Select Nova Spark. Nova's purple/silver theme and inventor intro appear.
- [ ] Select Dash Rocket. Dash's green/yellow theme and rocket intro appear.
- [ ] Every intro shows the correct name, type, icon, intro line, and catchphrase.
- [ ] Character portrait cards bob and themed particles float.

## Flow Checks

- [ ] Start Adventure goes to the Academy Hub.
- [ ] Choose Again returns to Character Select.
- [ ] Selecting a new character shows their intro again.
- [ ] The Academy Student Badge popup appears on the first intro only.
- [ ] Missing voice audio does not crash or interrupt the game.

## Maize Test

- [ ] A 4-year-old can pick a favorite character.
- [ ] The child understands that the character is saying hello.
- [ ] The child identifies Start Adventure as the main action.
- [ ] The child reaches the Academy Hub without adult help.
